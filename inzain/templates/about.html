About me

You can reach me via email at: zain@inzain.net.

About this site

The server is running Apache 2 (mod_wsgi) behind nginx. It is hosted by Slicehost on a 256mb Ubuntu 9.10 slice. The back-end is powered by PostgreSQL 8 and caching is done by memcached.

The blog is a custom Django application that I've put on GitHub. Code highlighting is done by pygments, search is powered by Haystack, and tweets are grabbed using python-twitter.
